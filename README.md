# Computational Physics Toolkit

A running collection of computational physics projects, exercises, and mini-implementations as I learn and document new topics. Each subfolder is a self-contained project with its own README, code, and (where relevant) figures.

## Contents

- [ising-model](./ising-model): 2D Ising model simulation using the Metropolis-Hastings MCMC algorithm.
- - [`monte-carlo`](./monte-carlo) — Monte Carlo integration and variance reduction techniques (importance sampling, control variates, antithetic variables).

- More projects will be added over time.
